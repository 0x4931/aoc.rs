## User Guide

To build binaries for every day of every year, run:

```bash
$ cargo build
```

To build binaries for a specific day and year, run:

```bash
$ cargo build --bin YYYY_DD
```

To run a binary for a specific day and year, run:

```bash
$ cargo run --bin YYYY_DD
```
